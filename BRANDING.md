# SpinSnap Arcade 브랜딩 가이드

## 🎨 브랜드 개요

**SpinSnap Arcade**는 물리 엔진 기반의 인터랙티브 게임 플랫폼입니다.

### 브랜드 컨셉

- **핵심 가치**: 재미, 물리 법칙, 상호작용
- **타겟**: 모든 연령대의 캐주얼 게이머
- **특징**: 브라우저에서 바로 즐기는 무료 게임

## 🎮 게임별 브랜딩

### 플링코 (Plinko)

- **색상**: Cyan (#06b6d4)
- **아이콘**: 주사위/격자 패턴
- **키워드**: 운, 확률, 행운

- **색상**: Green (#10b981)
- **아이콘**: 트로피/삼각형
- **키워드**: 속도, 경쟁, 스릴

### 서바이벌 (Survival)

- **색상**: Purple (#9333ea)
- **아이콘**: 해골/원형 링
- **키워드**: 생존, 전략, 배틀로얄

### 컬러 컨퀘스트 (Color Conquest)

- **색상**: Pink (#ec4899)
- **아이콘**: 팔레트/여러 색상의 원
- **키워드**: 영토, 전략, 색칠

## 📁 파일 구조

```
public/
├── favicon.svg          # 벡터 파비콘
├── favicon.ico          # 레거시 파비콘
├── icon-192.png         # PWA 아이콘 (작음)
├── icon-512.png         # PWA 아이콘 (큼)
├── apple-touch-icon.png # iOS 홈 화면 아이콘
├── og-image.svg         # 소셜 미디어 이미지
├── manifest.json        # PWA 매니페스트
└── robots.txt           # 검색 엔진 크롤러 설정

src/app/
├── layout.tsx           # 루트 레이아웃 (글로벌 메타데이터)
├── sitemap.ts           # 동적 사이트맵
└── [game]/
    └── layout.tsx       # 게임별 메타데이터
```

## 📝 사용 예시

### 환경 변수 설정

```bash
# .env.local 파일 생성
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

### 이미지 변환

실제 프로덕션에서는 SVG를 PNG로 변환해야 합니다:

```bash
# ImageMagick 사용 예시
convert -background none -resize 192x192 favicon.svg icon-192.png
convert -background none -resize 512x512 favicon.svg icon-512.png
convert -background none -resize 180x180 favicon.svg apple-touch-icon.png

# 또는 온라인 도구 사용
# https://cloudconvert.com/svg-to-png
# https://convertio.co/svg-png/
```
